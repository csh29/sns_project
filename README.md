# sns_project

## 👨‍🏫 프로젝트 소개
NextJS를 기반으로 한 SNS 애플리케이션입니다. NextJS와 nodeJS공부 목적으로 제작.

## 💻 개발환경
- **Next** : 12 Version
- **state-lib** : redux-saga, redux/toolkit,   
- **Node.js**
- **Mysql**

## 📌 주요 기능
- 소셜로그인
    - 카카오에서 제공하는 api를 통해 간편 로그인 구현
